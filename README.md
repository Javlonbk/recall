# recall

**Live → [recalldev.pages.dev](https://recalldev.pages.dev)**

*What you can't recall, you never knew.*

- **Q&A** — the primary surface: question → answer, collapse-to-recall, search, review.
- **Book** — long-form reference per subject.

Subjects: JavaScript, TypeScript, Node.js, System Design.

## Run locally

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static output → dist/
```

Built with [Astro](https://astro.build) as a fully static site.
