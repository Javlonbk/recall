# CLAUDE.md — recall

## What this repo is

A personal study-notes website. Topics start with JavaScript, then TypeScript, Node, NestJS, Postgres, React, system design.

Deployed as a static site (free subdomain). The repo name `recall` is the point: the site is built around **active recall**, not re-reading.

## Structure (per topic)

Two parts:

1. **The book** — full reference. Searchable, deep, long-form. Used when the owner needs detail, not for primary study.
2. **Q&A** — the primary surface. A question-led notes format: every concept is framed as a **question**, answered in the **clearest minimal form** — a table, code, a diagram, or one or two sharp lines.

Not a quiz. No scoring, no "did you get it right". The question framing exists to make ideas instantly clear and easy to revise — the opposite of long boring docs. Answers must look great: clean tables, syntax-highlighted code, diagrams.

> The Q&A is the spine; the book is reference.
